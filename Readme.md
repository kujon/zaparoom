## Requirements
* Node (https://nodejs.org/)
* Heroku Toolbelt (https://toolbelt.heroku.com/)

## Setup
### Dependencies
```
npm install
```
### Tests
```
gulp test
```
### Linting
```
gulp lint swag
```
### Build
```
gulp bundle
```
### Run
```
npm start
```
